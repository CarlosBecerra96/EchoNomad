# EchoNomad 🌍✈️

EchoNomad es un proyecto web desarrollado como entrega para el curso de Desarrollo Web en CoderHouse.  
El sitio refleja mi pasión por viajar y está compuesto por varias secciones que muestran destinos, herramientas de planeación, reflexiones en un blog y un formulario de contacto.

## 🚀 Tecnologías utilizadas
- **HTML5** para la estructura del sitio
- **SCSS/CSS3** para estilos y modularidad
- **Bootstrap 5** para componentes y grillas responsivas
- **GitHub Pages** para la publicación online

## 📂 Estructura del proyecto
- `index.html` → Página principal
- `pages/` → Páginas secundarias (Destinos, Planeación, Blog, Contacto)
- `css/` → Estilos compilados desde SCSS
- `img/` → Imágenes y recursos gráficos

## ✨ Características
- Navbar y footer consistentes en todas las páginas
- Secciones **hero** con imágenes de fondo y overlay
- Cards responsivas para destinos, herramientas y publicaciones
- Formulario de contacto estilizado con Bootstrap
- Diseño limpio y modular
