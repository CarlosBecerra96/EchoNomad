# EchoNomad 🌍✈️

[🔗 Ver proyecto en Vercel]

---

## 📖 Descripción
*EchoNomad* es un proyecto web desarrollado como entrega final del curso de Desarrollo Web.  
Su objetivo es inspirar a los viajeros a explorar destinos únicos, planificar aventuras y compartir experiencias a través de un diseño moderno y responsivo.

---

## 🚀 Tecnologías utilizadas
- *HTML5* para la estructura del contenido.
- *SCSS/Sass* para estilos modulares y mantenibles.
- *CSS3* compilado desde Sass.
- *Bootstrap 5* para componentes y utilidades responsivas.
- *Git & GitHub* para control de versiones.
- *Vercel* para despliegue en producción.

---

---

## ✨ Funcionalidades principales
- *Hero section* con imagen de fondo, títulos grandes y botón estilizado.  
- *Navegación clara* con secciones: Home, Destinos, Planeación, Blog y Contacto.  
- *Botón personalizado* con borde blanco, relleno transparente y hover con color de acento.  
- *Diseño responsivo* adaptable a móviles, tablets y escritorio.
